# Git installation
 
It is easiest to install Git on Linux with your distribution's package manager.


## Debian/Ubuntu

```bash
apt-get install git
```

## Fedora 22 and later

```bash
dnf install git
```

## Gentoo

```bash
emerge --ask --verbose dev-vcs/git
```

## Arch Linux

```bash
pacman -S git
```

## openSUSE
```bash
zypper install git
```

## Mageia

```bash
urpmi git
```

## Nix/NixOS

```bash
nix-env -i git
```

## FreeBSD

```bash
pkg install git
```

## Solaris 9/10/11 (OpenCSW)

```bash
pkgutil -i git
```

## Solaris 11 Express, OpenIndiana

```bash
pkg install developer/versioning/git
```
## OpenBSD

```bash
pkg_add git
```

## Alpine

```bash
apk add git
```
